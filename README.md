# 📋 Aplikasi Pengingat Tugas

Aplikasi web modern untuk mengelola tugas harian Anda dengan mudah dan efisien.

## 🚀 Fitur-Fitur

✅ **Tambah Tugas** - Tambahkan tugas baru dengan cepat
✅ **Tandai Selesai** - Centang tugas yang sudah dikerjakan
✅ **Hapus Tugas** - Buang tugas yang tidak diperlukan
✅ **Filter Tugas** - Tampilkan tugas berdasarkan status (Semua, Aktif, Selesai)
✅ **Bersihkan Otomatis** - Hapus semua tugas yang sudah selesai sekaligus
✅ **Penyimpanan Otomatis** - Data tersimpan di browser (localStorage)
✅ **Desain Responsif** - Bekerja sempurna di desktop, tablet, dan mobile
✅ **Notifikasi** - Notifikasi visual untuk setiap aksi
✅ **Progressive Web App** - Bisa diinstall seperti aplikasi native
✅ **Offline Support** - Tetap bisa digunakan tanpa internet
✅ **Tanpa Internet** - Semua data disimpan lokal, tidak perlu koneksi

## 📦 Instalasi

### 💻 Di Komputer
1. **Clone atau Download** folder ini ke komputer Anda
2. **Buka file** `index.html` di browser favorit Anda
   - Klik dua kali pada file `index.html`, atau
   - Drag & drop ke tab browser, atau
   - Klik kanan → Buka dengan → Pilih browser

### 📱 Di HP/Mobile
Aplikasi ini adalah **Progressive Web App (PWA)** yang bisa diinstall langsung di HP!

**Android (Chrome):**
1. Buka file `index.html` di Chrome mobile
2. Tekan menu ⋮ → "Install app"
3. Aplikasi langsung ada di home screen

**iPhone (Safari):**
1. Buka file di Safari
2. Tap Share → "Add to Home Screen"
3. Selesai!

📖 **Lihat panduan detail di file [INSTALL_MOBILE.md](INSTALL_MOBILE.md)**

## 💻 Cara Menggunakan

### Menambah Tugas
1. Ketik tugas Anda di kolom input "Masukkan tugas baru..."
2. Klik tombol **"+ Tambah"** atau tekan **Enter**
3. Tugas akan langsung muncul di daftar

### Menandai Tugas Selesai
1. Centang kotak checkbox di sebelah tugas
2. Tugas akan berubah menjadi warna lebih gelap dengan garis coret

### Menghapus Tugas
1. Klik tombol **"🗑️ Hapus"** di sebelah tugas yang ingin dihapus
2. Tugas akan dihapus langsung dari daftar

### Filter Tugas
- **Semua** - Tampilkan semua tugas
- **Aktif** - Hanya tampilkan tugas yang belum selesai
- **Selesai** - Hanya tampilkan tugas yang sudah selesai

### Bersihkan Tugas Selesai
1. Klik tombol **"🗑️ Bersihkan Tugas Selesai"** (muncul jika ada tugas selesai)
2. Konfirmasi pemberian

## 📊 Statistik

Aplikasi secara otomatis menghitung dan menampilkan:
- **Semua** - Total seluruh tugas
- **Aktif** - Jumlah tugas yang belum selesai
- **Selesai** - Jumlah tugas yang sudah diselesaikan

## 💾 Penyimpanan Data

Semua tugas Anda disimpan secara otomatis di **localStorage** browser. Ini berarti:
- ✅ Data tetap tersimpan meski browser ditutup
- ✅ Data tersimpan per browser (bukan di cloud)
- ✅ Tidak perlu setup database
- ⚠️ Data akan hilang jika cache browser dihapus
